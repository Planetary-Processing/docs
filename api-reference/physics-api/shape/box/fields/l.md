# L

`box.L`

The length of the bounding box, in world units along the Y axis.

| Type    | Initialised Value  | Description             |
| ------- | ------------------ | ----------------------- |
| `float` | eg. `2.4`          | The length of the box.  |



Example:

Increase the physical length of two cats every time they collide.
