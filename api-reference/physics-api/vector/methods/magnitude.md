# Magnitude

`vector:Magnitude()`

Returns the magnitude of a vector. Square roots the sum of the individually squared X, Y, and Z values, to return a single numerical value.

The magnitude is the length of the vector, if measured as a distance in 3D space.



Returns:

| Type    | Description                                                                                                                                                                                                                                           |
| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `float` | <p>The scalar magnitude of a vector.<br><br>Eg. The vector which called the function (a) finding the root of its squared values summed :</p><p><code>math.sqrt(</code><br><code>a.X^2 â€‹</code> <br><code>+ a.Y^2</code> <br><code>+ a.Z^2)</code></p> |
