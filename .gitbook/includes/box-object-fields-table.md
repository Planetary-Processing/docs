title: Box Object Fields Table

| Name | Type    | Description                                         |
| ---- | ------- | --------------------------------------------------- |
| W    | `float` | The world measurement of the box along the X axis.  |
| L    | `float` | The world measurement of  the box along the Y axis. |
| D    | `float` | The world measurement of  the box along the Z axis. |
